## Features
**Fully Offline** ‚Äî No cloud uploads, all conversions happen locally  
**Multi-Format Support** ‚Äî `.doc`, `.docx`, `.ppt`, `.pptx`, `.pdf`  
**Smart Conversion** ‚Äî Tries Microsoft Office COM first, falls back to LibreOffice if available  
**Batch Processing** ‚Äî Convert and merge an entire folder in one go  
**Clean Modern UI** ‚Äî File table view, progress bar, status updates  
**Cross-Platform** ‚Äî Works on Windows, macOS, and Linux

## Requirements

You‚Äôll need:
- **Python 3.8+** installed and added to PATH
- **Required Python packages** (see [Installation](#-installation))
- **One of the conversion backends**:
  - **Microsoft Office (Word & PowerPoint)** ‚Äî *Windows only*  
  - **LibreOffice** ‚Äî cross-platform, install from [libreoffice.org](https://www.libreoffice.org/)

## Installation

**1. Clone or download** this repository:
```bash
git clone https://github.com/whoravinder/addtopdf
cd addtopdf
```
**2. Installing Requirements** :
```bash
pip install requirements.txt
```
**3. Installing Requirements** :
Open Command Prompt(win+r->cmd) and navigate to directory containing `main.py` file and then type in following command 
```bash
python main.py
```
