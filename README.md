## Features
**Fully Offline** ‚Äî No cloud uploads, all conversions happen locally  
**Multi-Format Support** ‚Äî `.doc`, `.docx`, `.ppt`, `.pptx`, `.pdf`  
**Smart Conversion** ‚Äî Tries Microsoft Office COM first, falls back to LibreOffice if available  
**Batch Processing** ‚Äî Convert and merge an entire folder in one go  
**Clean Modern UI** ‚Äî File table view, progress bar, status updates  
**Cross-Platform** ‚Äî Works on Windows, macOS, and Linux
